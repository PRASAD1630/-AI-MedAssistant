# 🩺 AI-MedAssistant – Smart Medical Assistant using LLMs & RAG

AI-MedAssistant is a full-stack intelligent healthcare assistant that leverages **Large Language Models (LLMs)** and **Retrieval-Augmented Generation (RAG)** to provide context-aware medical information through an interactive web interface. The application combines modern AI technologies with a Flask backend to simulate a smart healthcare support system capable of answering medical queries, managing appointments, and maintaining patient records.

> **Note:** This repository is maintained for educational purposes and AI application development practice.

---

## 🚀 Features

- 🤖 AI-powered medical chatbot using Large Language Models
- 📚 Retrieval-Augmented Generation (RAG) for context-aware responses
- 💬 Interactive chat interface for healthcare assistance
- 👨‍⚕️ Patient, doctor, and appointment management
- 📅 Appointment scheduling and modification
- 🧠 Semantic search using FAISS vector database
- 🗄️ SQLite database for patient records and chat history
- 📄 Downloadable conversation history
- 🚨 Emergency reporting with priority classification
- 🔒 Modular and scalable application architecture

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/PRASAD1630/-AI-MedAssistant.git
```

### Navigate to the project

```bash
cd AI-MedAssistant
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Download the LLM

```bash
ollama pull llama3.1
```

### Build the vector database

```bash
python src/create_index.py
```

### Initialize the database

```bash
python src/create_db.py
```

### Run the application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:8080
```
